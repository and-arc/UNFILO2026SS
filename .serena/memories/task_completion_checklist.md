# タスク完了後の確認事項

## デプロイ前チェックリスト

### 1. コード品質確認
- [ ] SCSSが正常にコンパイルされているか
- [ ] JavaScript構文エラーがないか
- [ ] HTMLバリデーション通過
- [ ] 画像の最適化確認

### 2. 機能動作確認
- [ ] スムーズスクロール動作
- [ ] レスポンシブ表示（768px境界）
- [ ] 固定ナビゲーション開閉
- [ ] アンカーリンク遷移
- [ ] 外部リンク動作（target="_blank"）
- [ ] ページトップボタン

### 3. ブラウザ横断確認
- [ ] Chrome最新版
- [ ] Safari最新版
- [ ] Firefox最新版
- [ ] スマートフォンブラウザ（iOS Safari、Android Chrome）

### 4. パフォーマンス確認
- [ ] ページ読み込み速度
- [ ] 画像読み込み最適化
- [ ] CSSファイルサイズ

### 5. SEO・アナリティクス
- [ ] metaタグ設定確認
- [ ] OGP設定確認
- [ ] GTMタグ動作確認
- [ ] Canonical URL設定

### 6. アクセシビリティ
- [ ] alt属性の適切な設定
- [ ] セマンティックなマークアップ
- [ ] キーボードナビゲーション

## 公開手順
1. 最終動作確認完了
2. 本番サーバーへファイルアップロード
3. 公開URL動作確認
4. アナリティクス計測開始確認
5. ステークホルダーへ公開通知

## 保守・更新時の注意点
- 商品情報更新時は該当HTMLセクションとSCSSを同時確認
- 画像更新時はファイル名規則に従う
- 季節・キャンペーン変更時は全体的なテキスト・画像見直し
- アナリティクス計測IDは環境に応じて切り替え
